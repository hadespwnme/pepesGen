## How to run 🖥️️
1. Clone the repo
   ```sh
   git clone https://github.com/yingcrackerhades/pepesGen
   ```
2. Entry the path
   ```sh
   cd pepesGen
   ```
3. Run app
   ```sh
   python3 pepesGen.py
   ```
